SITREP Visual & Threat Map Creation

Buka Sesi TEKNIK

SITREP Visual Components

Header/DTG
Threat Level Indicator
Enemy Status Card
Friendly Status Card
Key Events Timeline
Action Items
SITREP FORMAT
1. SITUATION (Enemy/Friendly/Environment)
2. ASSESSMENT (Overall evaluation)
3. ACTIONS (Taken + Required)
4. NEXT UPDATE (Scheduled time)

Threat Map Elements

Hostile positions (red markers)
Friendly positions (blue markers)
Areas of interest (amber zones)
Movement corridors (dashed lines)
Threat gradient (heat map)
Key Principle

Map harus bisa dibaca dalam 3 detik — threat level harus immediately obvious.

PROMPT TEMPLATES

VISUAL CREATION
PROMPT LIBRARY

TEMPLATE 01 • Situational Dashboard
Buatkan dashboard situational awareness dengan data berikut: DATA INPUT: [PASTE: threat level, asset count, key metrics] KOMPONEN YANG DIBUTUHKAN: 1. HEADER: DTG dan overall threat level (color-coded) 2. KPI CARDS (3-4 cards): - Total assets monitored - Active threats - Operational status percentage 3. ACTIVITY CHART: Timeline 24 jam terakhir 4. ALERT PANEL: Top 3 priority items STYLE REQUIREMENTS: - Dark theme (navy background) - Color coding: Red=critical, Amber=warning, Green=normal - Font: Clean, readable, military-style - Layout: Grid-based, responsive OUTPUT: Single-page dashboard layout dengan semua komponen di atas.
TEMPLATE 02 • Threat Map
Buatkan threat map visualization untuk area operasi: AREA: [REGION/SECTOR NAME] ELEMENTS TO PLOT: HOSTILE POSITIONS (Red markers): [LIST: location, type, threat level] FRIENDLY ASSETS (Blue markers): [LIST: location, type, status] AREAS OF INTEREST (Amber zones): [LIST: area name, reason for interest] MAP REQUIREMENTS: - Clear legend with all marker types - Scale indicator - North arrow - Sector boundaries - Movement corridors (if applicable) - Heat map overlay for threat density OUTPUT: Tactical map ready for command briefing.
TEMPLATE 03 • SITREP Visual
Transform SITREP teks berikut menjadi visual summary: SITREP DATA: [PASTE SITREP TEXT] VISUAL FORMAT: 1. HEADER SECTION - DTG (prominent) - Overall threat level indicator (large, color-coded) - Reporting unit 2. STATUS CARDS (side-by-side) - ENEMY: Key activities, strength, disposition - FRIENDLY: Status, casualties, readiness - ENVIRONMENT: Weather, terrain factors 3. KEY EVENTS TIMELINE - Chronological list with timestamps - Icons for event types 4. ACTION ITEMS - Completed actions (checked) - Required actions (highlighted) - Recommendations DESIGN: Professional military briefing style, scannable in under 10 seconds.
TEMPLATE 04 • Intel Summary Infographic
Buatkan infografis satu halaman untuk intelligence summary: INTELLIGENCE BRIEF: [PASTE INTEL SUMMARY] INFOGRAPHIC SECTIONS: 1. HEADLINE (Top) - Main finding in 1 sentence - Threat level badge 2. KEY FINDINGS (Icons + Short text) - Finding 1: [icon] + 10 words max - Finding 2: [icon] + 10 words max - Finding 3: [icon] + 10 words max 3. THREAT MATRIX (2x2 grid) - Capability vs Intent visualization - Position of threat actor marked 4. CONFIDENCE METER - Overall confidence: HIGH/MEDIUM/LOW - Visual indicator (gauge or bar) 5. RECOMMENDATIONS (Bottom, highlighted) - 3 bullet points max - Priority indicated STYLE: Clean, professional, high-contrast for projection.
