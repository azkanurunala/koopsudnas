BRIEFING PACKAGE PROMPT

📋 COMPLETE BRIEFING PACKAGE PROMPT
Buatkan complete command briefing package dengan 4 HTML pages/sections:

=== OPERATIONAL DATA ===
DTG: [121500Z APR 2026]
Assessment Period: [Last 48 hours]
Classification: [UNCLASS/SIMULATED]

SITUATION:
- Threat Level: [HIGH]
- Total Contacts: [62] in [4] sectors
- Hot Sector: [NE-7] with [+300%] activity
- Sources: [SIGINT confirmed, IMINT partial, HUMINT B-2]

TREND DATA (hourly, last 48h):
[5,6,8,10,12,15,18,22,28,32,38,45,52,58,62,60,...]

FRIENDLY STATUS:
- Air Defense: [READY]
- CAP: [STANDBY - 2 aircraft]
- Radar: [100% coverage]

=== CREATE 4 SLIDES ===

SLIDE 1 - EXECUTIVE DASHBOARD:
- Threat level badge (large, prominent)
- Key metrics (contacts, sectors, confidence)
- BLUF statement
- DTG/period

SLIDE 2 - THREAT MAP:
- Grid with sector labels
- Red markers = hostile, Blue = friendly
- Highlighted threat zone
- Legend

SLIDE 3 - TREND ANALYSIS:
- Bar/line chart showing 48h trend
- Color gradient (green→amber→red)
- Baseline comparison
- Trend indicator (escalating/stable/declining)

SLIDE 4 - RECOMMENDATIONS:
- Immediate actions (0-6h)
- Short-term actions (24-72h)
- Decision point criteria
- Next update time

=== DESIGN REQUIREMENTS ===
- Dark background (navy #050810)
- Consistent color coding across all slides
- Each slide fits single viewport
- Professional military aesthetic
- 3-second rule compliance
- Print-friendly (white text readable)

OUTPUT: 4 complete HTML sections with inline CSS

GEMINI WORKFLOW • SESI 6.3 CANVAS

Briefing Narrative Document — Draft dengan Section Editing
KAPAN PAKAI UNTUK COMMAND BRIEFING

Briefing komandan tingkat Pangkoopsud selalu diiringi briefing document tertulis (backgrounder, talking points, Q&A prep). Canvas adalah workspace terbaik untuk draft document ini karena Anda bisa iterate setiap section (Background, Key Findings, Recommendations, Q&A) secara terpisah saat slide ditune.

STEP-BY-STEP — Briefing Document di Canvas
  1. 1
    Minta skeleton briefing document — Cover page, BLUF, Background, Analysis, Options/COAs, Recommendation, Q&A prep, Appendix.
  2. 2
    Isi BLUF (Bottom Line Up Front) dulu — ini paling penting, maksimal 3 kalimat, komandan baca ini first.
  3. 3
    Q&A Prep Section — minta AI generate 10 kemungkinan pertanyaan komandan + jawaban pendek. Critical untuk persiapan.
  4. 4
    Iterate saat slide di-refine — setiap perubahan slide → update section terkait di document, preserve lain.
  5. 5
    Export ke Google Docs → print sebagai hand-out untuk komandan dan staf.
PROMPT SIAP PAKAI — Briefing Document + Q&A Prep
Buat Canvas briefing document lengkap untuk presentasi ke Pangkoopsud.

Topik: [ISI TOPIK, e.g., "Threat Assessment Kontak Unidentified Sektor Timur"]
Durasi briefing: 15 menit
Audience: Pangkoopsud + staf inti

Struktur:

1. COVER PAGE
   - Judul
   - Classification banner
   - Prepared by, Date, Distribution

2. BLUF (3 kalimat maksimal)
   - What happened / threat level
   - What we're doing about it
   - What decision we need from commander

3. BACKGROUND (1 paragraf)
   - Konteks situasional singkat
   - Timeline kejadian

4. KEY FINDINGS (3-5 bullet dengan confidence)
   - Setiap finding: observed fact + assessment + confidence

5. ANALYSIS (2-3 paragraf)
   - Capability, intent, opportunity
   - Historical analog (jika ada)
   - Data gaps

6. OPTIONS / COA (2-3 courses of action)
   - Per COA: description, pros, cons, resource requirement

7. RECOMMENDATION
   - COA preferred + justification
   - Implementation timeline
   - Success criteria

8. Q&A PREP — PENTING
   Generate 10 likely questions dari Pangkoopsud dengan jawaban pendek:
   - Questions about capability
   - Questions about intent
   - Questions about resource/cost
   - Questions about timeline
   - Questions about alternative scenarios
   - Questions about worst case
   - Questions about precedent
   - Questions about decision authority
   - Questions about escalation criteria
   - Questions about what to tell media/public

9. APPENDIX
   - Sources used
   - Methodology note
   - Confidence matrix

Format: formal military Indonesia, siap cetak A4, tone professional untuk level Pangkoopsud.
💡 Tips Briefing Document
  • • BLUF selalu pertama — komandan sering baca BLUF, skip sisanya. Buat 3 kalimat yang standalone.
  • • Q&A prep critical — 80% kesuksesan briefing = persiapan Q&A. Generate 10 question, jawab all.
  • • Recommendation < 1 paragraph — clear, actionable, dengan success criteria.
  • • Document ≠ slide content — document lebih detail, slide lebih visual. Jangan kopi-paste.

GEMINI WORKFLOW • SESI 6.5 CREATE VIDEO

Video Brief Element — Intro Video + Terrain Visualization
KAPAN PAKAI UNTUK COMMAND BRIEFING

Briefing modern butuh elemen visual yang immersive — bukan sekadar slide teks. Video 5-8 detik cocok untuk: intro video (establishing atmosphere sebelum BLUF), terrain visualization (menunjukkan lingkungan ops secara generic), atau skenario ilustrasi (menampilkan konteks situasional). Veo menghasilkan clip berkualitas cinematic dari deskripsi teks dalam 2-3 menit.

STEP-BY-STEP — Video untuk Briefing
  1. 1
    Identifikasi 2-3 video moments di briefing flow — intro video (slide 1), terrain establishing (slide 3), dan scenario illustration (slide 7).
  2. 2
    Generate tiap clip terpisah via Create Video — satu per satu, generate 2-3 variasi per clip, pilih yang terbaik.
  3. 3
    Download MP4 clips — simpan di folder briefing package.
  4. 4
    Insert ke Google Slides → Insert > Video > Upload. Atur timing dan auto-play jika perlu.
  5. 5
    Gabungkan dengan audio dari Create Music (Embed 3) — video + audio = immersive briefing element.

💡 Tips Video untuk Briefing
  • • 5-8 detik cukup — panjang = distraksi. Video brief harus complement narrative, bukan steal attention.
  • • Generate 2-3 variasi per clip, pilih yang paling professional.
  • • Auto-play di Slides — set "play automatically" supaya flow tidak terganggu klik.
  • • Loop untuk establishing shot — clip pendek yang loop = background atmospheric tanpa audience sadar.
  • • Generic framing — hindari kata militer spesifik di prompt supaya lulus content filter.

GEMINI WORKFLOW • SESI 6.6 CREATE MUSIC

Briefing Audio Scoring — Background Music untuk Tone
KAPAN PAKAI UNTUK COMMAND BRIEFING

Background music di command briefing jarang dipakai di military context, tapi untuk briefing yang panjang, ceremonial, atau high-stakes, musik subtle bisa membantu mempertahankan atensi dan membangun tone. Gunakan HANYA untuk intro/outro dan transitions — bukan under narrative (akan distraksi).

STEP-BY-STEP — Briefing Audio Score
  1. 1
    Identifikasi moment untuk audio — intro (30s), section transition (5-10s), outro (30s). Bukan keseluruhan briefing.
  2. 2
    Tentukan mood — serious/authoritative untuk threat briefing, hopeful untuk SAR success, solemn untuk lesson-learned.
  3. 3
    Generate intro track via Create Music, 30 detik, tanpa lirik, loop-friendly ending.
  4. 4
    Download MP3, insert ke Slides via Insert > Audio, set auto-play untuk slide pertama.
  5. 5
    Fade-out saat narrative start — audio hanya untuk intro, stop sebelum komandan/briefer mulai bicara.
PROMPT SIAP PAKAI — Briefing Intro Score
Generate orchestral instrumental music for military command briefing intro.

Style: Cinematic, professional, serious
Mood: Authoritative, focused, disciplined — conveys importance without alarm
Instruments: Strings (dominant), low brass (subtle accent), minimal percussion
Tempo: Slow to moderate (60-75 BPM)
Structure:
- 0-5s: subtle opening (single string note)
- 5-15s: gradual buildup (add more strings, soft brass)
- 15-25s: sustained middle (full but controlled)
- 25-30s: gentle fade that loops cleanly

Duration: 30 seconds
NO vocals, NO lyrics, NO sudden loud hits
Tone: befitting a senior military officer presenting strategic information
Purpose: background music for briefing intro slide, will fade out when narrative starts
💡 Tips Briefing Audio
  • • Kurang > lebih — audio di briefing = risk distraksi. Gunakan hanya di intro/outro.
  • • Fade-out wajib — audio harus hilang sebelum narrative. Atur di Slides audio settings.
  • • Test di ruangan aktual — audio yang oke di headphone bisa terdengar buruk di speaker ruang briefing. Test dulu.
  • • Match mood ke topik — serius untuk threat, hopeful untuk SAR victory, solemn untuk lessons learned.
  • • Internal use = aman — AI-generated music OK untuk briefing internal; cek policy kalau video akan dipublikasikan eksternal.
