Deep Research: Konsep & Metodologi

Buka Sesi KONSEP

Apa itu Deep Research?

Deep Research adalah kemampuan AI untuk menganalisis multiple documents, mengekstrak informasi kunci, dan menyintesis insights yang koheren — semua dalam hitungan menit.

Kapabilitas Deep Research:
  • Cross-reference multiple documents
  • Identify contradictions & gaps
  • Extract temporal patterns
  • Generate confidence scores

Metodologi 4-Step

1
COLLECT
Kumpulkan semua sumber relevan
2
ANALYZE
Ekstrak informasi kunci per sumber
3
CORRELATE
Temukan hubungan antar sumber
4
SYNTHESIZE
Hasilkan unified intelligence picture

Hands-On: Threat Assessment dari Data Simulasi

Buka Sesi PRAKTIK

1 Guided Analysis (35 min)

Instruktur memandu analisis dataset simulasi radar:

  • 50 data points dari 24 jam monitoring
  • 3 sektor coverage (NE, E, SE)
  • Mixed signals: routine + anomalies

2 Independent Exercise (40 min)

Exercise: Complete Threat Assessment

Diberikan dataset baru (100 data points, 48 jam). Identifikasi:

  • • Minimum 3 patterns
  • • 1 high-priority anomaly
  • • Threat level assessment
  • • Recommended response
Tip: Iterative Prompting

Jangan expect perfect output dari satu prompt. Gunakan follow-up questions untuk drill down ke specific patterns.

SESI 4 • 60 MENIT

Multi-Source Intelligence Fusion

Buka Sesi ADVANCED

Sesi puncak modul: menggabungkan multiple intelligence sources menjadi unified assessment. Ini adalah skill yang membedakan analyst pemula dari expert.

SOURCE A
SIGINT Report
Increased comm traffic in sector NE
SOURCE B
Radar Data
Unusual flight patterns detected
SOURCE C
HUMINT Field Report
Local observation of military vehicles
Fusion Questions to Ask AI:
  • 1. "Apakah ada korelasi waktu antara SIGINT traffic increase dan radar anomaly?"
  • 2. "Bagaimana HUMINT report mengkonfirmasi atau kontradiksi technical intelligence?"
  • 3. "Apa hypothesis yang paling likely berdasarkan semua sumber?"
  • 4. "Apa intelligence gaps yang masih perlu diisi?"
TEMPLATE & FRAMEWORK

DEEP RESEARCH
PROMPT TEMPLATES

TEMPLATE 01 • Multi-Source Analysis
Kamu adalah [ROLE: intelligence fusion analyst] dengan expertise dalam multi-source analysis. TUGAS: Analisis dan sintesis informasi dari multiple sources berikut. === SOURCE 1: [SIGINT/IMINT/HUMINT/OSINT] === [PASTE DATA SOURCE 1] === SOURCE 2: [TYPE] === [PASTE DATA SOURCE 2] === SOURCE 3: [TYPE] === [PASTE DATA SOURCE 3] ANALISIS YANG DIPERLUKAN: 1. Per-source key findings 2. Cross-source correlations 3. Contradictions or gaps 4. Unified assessment 5. Confidence level per finding OUTPUT FORMAT: ## Executive Summary (100 kata) ## Source-by-Source Analysis ## Correlation Matrix ## Unified Intelligence Picture ## Confidence Assessment (HIGH/MEDIUM/LOW per finding) ## Intelligence Gaps & Collection Requirements
TEMPLATE 02 • Pattern Recognition
Kamu adalah [ROLE: pattern analysis specialist]. TUGAS: Identifikasi patterns dalam dataset operasional berikut. DATA INPUT: [PASTE: timestamp, location, event_type, value] JENIS PATTERN YANG DICARI: 1. TEMPORAL: Recurring events, frequency changes, timing anomalies 2. SPATIAL: Geographic clustering, movement corridors, hotspots 3. BEHAVIORAL: Tactic changes, response patterns, SOPs 4. CORRELATIONAL: Events that co-occur, cause-effect relationships UNTUK SETIAP PATTERN: - Description (what is the pattern?) - Evidence (which data points support it?) - Significance (why does it matter operationally?) - Confidence (how certain are we?) OUTPUT: ## Patterns Identified ### Pattern 1: [Name] - Type: [TEMPORAL/SPATIAL/BEHAVIORAL/CORRELATIONAL] - Description: ... - Supporting Data: [list data points] - Operational Significance: ... - Confidence: [HIGH/MEDIUM/LOW] ## Anomalies Detected ## Baseline Comparison ## Recommended Follow-up Analysis
TEMPLATE 03 • Comprehensive Threat Assessment
Kamu adalah [ROLE: threat assessment officer] di Koopsudnas. TUGAS: Lakukan comprehensive threat assessment berdasarkan intelligence berikut. INTELLIGENCE INPUTS: [PASTE ALL AVAILABLE INTEL] ASSESSMENT FRAMEWORK: 1. THREAT IDENTIFICATION - Who/What is the threat? - Known capabilities - Historical behavior 2. THREAT ANALYSIS - Current activity indicators - Intent assessment (what do they want to achieve?) - Capability vs Intent matrix 3. VULNERABILITY ASSESSMENT - Our exposed assets - Critical vulnerabilities - Time-sensitivity 4. RISK CALCULATION - Likelihood: [1-5] - Impact: [1-5] - Risk Score: [Likelihood x Impact] 5. RECOMMENDED RESPONSE - Immediate actions - Short-term measures - Long-term posture adjustment OUTPUT: Formal Threat Assessment Document - Classification: [UNCLASSIFIED/RESTRICTED] - DTG: [Date-Time Group] - Valid Until: [Expiry]
TEMPLATE 04 • Quick Intel Query (Iterative)
Berdasarkan data yang sudah kita diskusikan: [SPECIFIC QUESTION] Contoh follow-up questions: • "Fokus pada sektor NE saja, apa pattern yang terlihat?" • "Bandingkan aktivitas hari ini dengan baseline minggu lalu" • "Identifikasi 3 anomali paling signifikan" • "Apa yang mungkin terjadi dalam 24 jam ke depan?" • "Breakdown by time: pagi vs siang vs malam" • "Korelasikan dengan cuaca/kondisi visibility" Tips: Gunakan iterative questioning untuk drill down. Jangan ask everything in one prompt.
