5. Single viewport, no scroll
6. Responsive grid layout

VARIANT LINTAS DOMAIN

4 DASHBOARD + 4 INFOGRAFIS KOOPSUDNAS

Template prompt di atas (threat dashboard) adalah default untuk Operasi Pertahanan Udara. Di bawah ini 3 variant lain: SAR, Bencana, dan Readiness. Framework visual sama — hero metric + metric cards + chart + status — yang berubah: data, warna, stakeholder.

INFOGRAFIS VARIANT (EXECUTIVE SUMMARY)

Setiap dashboard di atas punya padanan infografis 1-halaman untuk executive summary: Threat Matrix → SAR Progress Matrix → Bencana Impact Matrix → Doctrine Compliance Trend. Struktur infografis (header + 3 key findings + matrix + recommendations + confidence meter) tetap sama. Ganti data & icon, warna mengikuti domain (merah/biru/hijau/ungu).

TOOL COMPARISON • ALTERNATIVE WORKFLOW

Gemini Canvas untuk Dashboard Interaktif

Dashboard dan infografis statis cukup dibuat di Claude Artifacts, tapi ketika Anda butuh live editor side-by-side dengan chat — Gemini Canvas jadi lebih cepat untuk iterate. Output-nya bisa HTML/SVG self-contained, siap di-host atau di-export.

CLAUDE ARTIFACTS
  • • Preview di panel kanan, live render
  • • Reasoning code lebih kuat (complex logic)
  • • Cocok untuk interactive tool dengan business logic
  • • React/TypeScript support
  • • Version history built-in
GEMINI CANVAS
  • • Fast iterate dengan inline edit
  • • SVG infografis crisp (teks akurat)
  • • Built-in data import dari Google Sheets
  • • Live document mode (bukan cuma code)
  • • Direct export ke Google Docs/Slides
🎯 Pilih Mana untuk Ops Koopsudnas?
  • • Interactive dashboard dengan CSV data → Canvas (import dari Sheets, quick viz)
  • • Prototype internal tool dengan logic kompleks → Claude Artifacts (reasoning lebih kuat)
  • • Infografis untuk briefing deck → Canvas SVG (lebih crisp), atau Imagen (illustrative)
  • • Live document draft briefing → Canvas (export langsung ke Docs)
