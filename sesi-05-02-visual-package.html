📋 VISUAL PACKAGE TEMPLATE
Create a visual intelligence package with 3 HTML deliverables:

=== INTELLIGENCE DATA ===
Operation: GARUDA SHIELD
DTG: 121500Z APR 2026
Threat Level: HIGH

Key Metrics:
- Aircraft contacts: 47 (+280%)
- Communications surge: 340%
- Naval vessels: 3 (AIS off)
- Ground personnel: ~50

Location Focus:
- Primary: NE-7 (23 contacts)
- Secondary: NE-8, NE-9
- Ground activity: NE-8 coastal

Timeline: Exercise 15-22 APR

=== CREATE 3 VISUALS ===

VISUAL 1: SITUATIONAL DASHBOARD
- Threat level badge (large, prominent)
- 4 key metric cards
- 48h activity trend chart
- Sector status overview

VISUAL 2: THREAT MAP
- 2x2 grid (NE-6, NE-7, NE-8, E-1)
- Red markers for air contacts
- NE-7 highlighted as threat zone
- Ground activity marker in NE-8
- Legend

VISUAL 3: SITREP VISUAL
- Header with DTG
- Threat level badge
- Enemy status section
- Friendly status section
- Recommended actions

=== DESIGN REQUIREMENTS ===
- Dark background (#050810)
- Consistent color coding
- 3-second rule per visual
- Professional military aesthetic
