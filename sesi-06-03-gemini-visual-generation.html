IMAGEN 4

STATIC IMAGE GENERATION

Imagen 4 adalah model image generation Google — built-in di gemini.google.com, tidak perlu tool terpisah. Cocok untuk:

✓ Untuk Ops Koopsudnas

  • • Sitrep visual — map-style situational overview
  • • Schematic diagram — force disposition, chain of command
  • • Infografis — statistik threat assessment
  • • Training visuals — ilustrasi untuk briefing
  • • Icon/symbol — unit marker, equipment silhouette

❌ Bukan untuk

  • • Faithful reproduction dari foto nyata (bisa halusinate detail)
  • • Personel identifiable (wajah spesifik)
  • • Real-world deception / disinformation
  • • Sensitive location dengan detail akurat
  • • Classified equipment dengan spec yang leak
⚠ OPSEC Gate sebelum prompt

Setiap request image generation harus lewat pertanyaan: "Jika orang luar melihat output ini, apakah membocorkan informasi operasional?" Kalau YA → jangan generate, atau generalize.

🎯 Prompting Framework: SUBJECT + STYLE + COMPOSITION + CONSTRAINT

Prompt image harus 4 komponen untuk reproducible, professional output.

SUBJECT: Tactical situational awareness infographic
STYLE: Modern military briefing, dark navy background, amber + cyan accent colors, flat vector illustration style
COMPOSITION: Top-down map view, 16:9 landscape, labeled zones with callout lines, legend in bottom-right corner
CONSTRAINT: No specific country names, generic aircraft silhouettes only, no identifiable personnel, no classified markings. Professional, non-sensationalized tone.

GEMINI CANVAS

INTERACTIVE DASHBOARDS

Canvas adalah fitur live editor di gemini.google.com — Gemini generate HTML/CSS/JS interactive di sebelah chat, Anda bisa iterate tanpa switch tool. Cocok untuk:

📊

Interactive Dashboard

Upload CSV data (threat indicators, sortie counts, equipment status) → "buatkan dashboard dengan chart trend + summary cards"

Output: HTML file self-contained, bisa di-host di local
📝

Live Document

Generate dokumen briefing — formal TNI format, tabel, heading auto. Lebih baik dari chat-to-doc karena live preview.

Output: Google Docs / PDF export langsung
🌐

Web App Prototype

Quick prototype untuk tools internal — threat tracker, intel log viewer, rapid form. No coding skill needed.

Output: functional HTML prototype
🎨

Infographic Builder

Generate infografis dengan SVG — lebih crisp dari Imagen untuk data viz yang butuh teks akurat

Output: SVG export, editable

💡 Canvas vs NotebookLM vs Claude

Canvas:Live interactive output — user klik, iterate, export. Ideal untuk dashboard & prototype.
NotebookLM:Source-grounded research. Bukan untuk build interactive output.
Claude Artifacts:Serupa Canvas — juga bisa build interactive HTML. Claude Artifacts umumnya lebih kuat di reasoning code-nya, Canvas lebih cepat iterate.
VEO

VIDEO GENERATION

🎬

Veo 3 — Google's Video Model

Generate video 8-60 detik dari text prompt atau image-to-video. Audio integration (dialogue, ambient) built-in sejak Veo 3.

⚠ Tersedia di Gemini Advanced (paid tier) dan AI Studio dengan quota terbatas.

✓ Legitimate Ops Use

  • • Training video — generic battlefield scenario untuk doctrine class
  • • Public affairs clip — hari TNI AU, ceremonial b-roll
  • • Concept visualization — show komandan "kira-kira seperti ini" untuk konsep operasi baru
  • • Recruitment material — visual appeal untuk generasi muda

❌ Red Flags

  • • Deepfake tokoh politik/militer — dilarang by policy
  • • Reconstruct insiden nyata — bisa dianggap disinformation
  • • Fake news narrative video — etis dan legal problem
  • • Classified visualization — equipment/facility yang bocor

🎯 Contoh Prompt Veo (acceptable)

SUBJECT: Generic military aircraft silhouette at sunset
CAMERA: Slow aerial tracking shot, cinematic, 24fps
MOOD: Professional, inspiring, not sensationalized
DURATION: 8 seconds, loop-ready
CONSTRAINT: No specific country markings, no identifiable squadron insignia, no real-world locations. Training material only.
OPSEC CRITICAL

VISUAL GENERATION RISKS

Visual generation tools lebih berbahaya dari text generation untuk OPSEC. Output visual bisa viral, di-screenshot, dan sulit di-retract. Sebelum generate apapun:

1
Classification Check
Apakah ada detail dalam visual yang jika dilihat publik akan membocorkan klasifikasi operasional? Contoh: aircraft spec, sensor package, facility layout.
2
Attribution Check
Jika visual keluar dari organisasi, apakah bisa diattribute ke Koopsudnas/TNI AU? Jika ya, pastikan ada watermark/disclaimer "ILUSTRASI — AI GENERATED".
3
Deception Check
Apakah visual bisa digunakan orang jahat untuk deceive atau spread disinformation? Terutama video Veo — generic sunset aircraft OK, tapi "reenactment" insiden spesifik berbahaya.
4
Retention Check
Output visual akan di-store di Google cloud. Cek kebijakan retention + data handling akun Anda. Data sensitif → jangan generate di akun personal.
5
Watermark Mandatory
Gemini menyisipkan SynthID watermark otomatis di semua output Imagen/Veo. Jangan crop/edit untuk hilangkan. Sebagai operator, tambahkan label "AI GENERATED • ILUSTRASI • [TANGGAL]" di setiap visual yang keluar organisasi.
EXERCISE

HANDS-ON 15 MENIT

🎯 Tasking

Generate infografis sitrep sederhana untuk brief komandan — visualisasi "Air Situation Awareness Indo-Pasifik Q1 2026" (generic, non-classified). Gunakan Imagen dulu, lalu coba rebuild dengan Canvas untuk perbandingan.

1
Tulis prompt 4-komponen (3 min)

Pakai framework Subject + Style + Composition + Constraint. Subject harus generic — tidak boleh country name, squadron, atau specific aircraft type.

2
Generate via Imagen (3 min)

Di gemini.google.com, paste prompt, select Imagen model. Generate 4 variant. Pick best one.

3
Rebuild di Canvas (5 min)

Prompt baru: "buatkan infografis SVG tentang [topic] dengan Gemini Canvas". Bandingkan hasil — mana yang lebih professional untuk slide briefing?

4
OPSEC Review (2 min)

Jalankan 5 OPSEC checks di slide sebelumnya. Apakah output Anda lolos? Jika tidak, iterate prompt untuk generalize lebih jauh.

5
Reflect (2 min)

Mana lebih cocok untuk produk Koopsudnas: Imagen (photoreal/ilustrasi) atau Canvas (SVG/structured)? Kenapa?
