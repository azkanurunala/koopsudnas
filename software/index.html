AI CODING TOOLKIT • 6 TOOLS

AI-Assisted Tools yang Dipakai di Course Ini

Setiap sesi di Modul 01-05 punya dedicated AI-Assisted Workflow slide yang menunjukkan cara pakai tools ini untuk topic sesi. Modul 06 jadi deep dive tool-by-tool.

🤖
GitHub Copilot
Autocomplete + chat IDE-native. Cepat untuk boilerplate, unit test, dan explain code di Visual Studio Code / JetBrains.
⚡
Cursor
IDE AI-first dengan Composer, inline edit, dan codebase-aware chat. Cocok untuk refactoring multi-file.
🧠
Claude Code
CLI agent dari Anthropic. Kuat untuk multi-step task, planning, skills, hooks, dan long-context codebase analysis.
🔧
Aider + Windsurf
Alternative terminal-based (Aider) dan IDE-native (Windsurf) agents. Bagus untuk git-native workflow.
🎨
v0 + Bolt
UI-first generation — dari prompt jadi component React/Next.js langsung running. Cocok untuk prototyping.
🔌
MCP Tooling
NEW
Model Context Protocol — cara standard untuk integrate tools, databases, dan APIs ke dalam AI coding agents.
